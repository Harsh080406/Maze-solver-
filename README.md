Developed an autonomous Maze Solver Robot capable of navigating complex maze environments and identifying the shortest path to the destination. The robot uses sensor-based obstacle detection and real-time decision-making algorithms to map its surroundings and avoid collisions. Built using Arduino/ESP32, ultrasonic/IR sensors, and motor drivers, the system demonstrates the integration of embedded hardware, robotics, and algorithmic problem-solving.

**Features**
Autonomous maze navigation
Shortest-path optimization
Sensor-based environment mapping
Efficient motor control and navigation

**Technologies Used**
ESP32
C++
8 Channel Array Sensor
TB612fng
Embedded Systems
Robotics

**Learning Outcomes**
Embedded systems programming
Sensor integration and calibration
Robotics control systems
Algorithm implementation
Hardware-software integration
Real-time problem solving
